# UniverseLink
Умноже сокращение ссылок с самоуничтожением и аналитикой.
Позволяет создавать короткие ссылки, которые можно ограничить по времени или количеству переходов.

## Технологии
**Фронтенд:**
- Vue 3
- Vite
- Tailwind CSS

## Установка и дальнейшая разработка

### Клонирование репозитория
```bash
git clone https://github.com/VladMontana/Frontend-Qbitlink.git
```

### Установка зависимостей 
```bash
npm install
npm run dev
```
